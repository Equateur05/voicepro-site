Support & FAQ

Trouvez les réponses à vos questions sur VoicePro

Prise en Main

Comment fonctionne VoicePro ?

+

VoicePro est une application de mastering audio professionnel pour iOS. Voici comment l'utiliser :

  • Importez votre fichier audio depuis votre appareil
  • Utilisez les outils professionnels pour affiner votre son (HPF, Gate, De-Ess, EQ, Compresseur, LUFS, Warmth, Limiter)
  • Exportez votre fichier mastered en WAV, MP3 ou AIFF
  • Partagez votre audio sur vos plateformes préférées

Tout le traitement audio se fait directement sur votre appareil, ce qui signifie que vos fichiers ne quittent jamais votre téléphone.

Quels formats audio sont supportés ?

+

VoicePro supporte tous les formats audio courants pour l'importation, notamment :

  • WAV
  • MP3
  • AIFF
  • AAC
  • M4A
  • FLAC
  • OGG

Vous pouvez exporter uniquement en WAV, MP3 et AIFF pour une qualité optimale.

Export & Partage

Comment exporter mon audio ?

+

Une fois que vous avez terminé le mastering de votre audio :

  • Appuyez sur le bouton "Exporter"
  • Choisissez votre format (WAV, MP3, ou AIFF)
  • Pour MP3, sélectionnez votre débit binaire préféré
  • Appuyez sur "Exporter" et sélectionnez où enregistrer le fichier
  • Vous pouvez ensuite partager directement depuis l'app ou accéder au fichier depuis l'app Fichiers

Quel format d'export devrais-je utiliser ?

+

WAV : Meilleure qualité, taille de fichier plus grande. Recommandé pour l'archivage et le traitement ultérieur.

MP3 : Compression avec perte, fichier plus petit. Parfait pour le partage en ligne et les réseaux sociaux.

AIFF : Qualité identique au WAV, format alternative. Idéal pour les workflows Pro Tools.

Puis-je partager directement depuis VoicePro ?

+

Oui ! Après avoir exporté votre fichier, vous pouvez le partager directement via :

  • Messages
  • Mail
  • Sauvegarde dans l'app Fichiers (pour accès à Dropbox, Google Drive, etc.)
  • Copie vers d'autres applications

Plan Premium

Combien coûte le plan Pro ?

+

Le plan Pro de VoicePro coûte 2,99 € pour un accès à vie.

Cet achat unique vous donne accès illimité à tous les outils professionnels de mastering sans frais supplémentaires.

Comment débloquer le plan Pro ?

+

Vous avez plusieurs options pour débloquer VoicePro Pro :

  • Essais gratuits : Vous avez accès à 3 essais gratuits pour tester tous les outils
  • Achat unique : Achetez le plan Pro à 2,99 € depuis l'app pour un accès permanent
  • Gestion des achats : Tous les achats in-app sont gérés par Apple et facturés à votre compte App Store

Qu'est-ce qui est inclus dans le plan Pro ?

+

Avec le plan Pro, vous avez accès à tous les outils professionnels :

  • HPF (High-Pass Filter)
  • Gate
  • De-Ess
  • EQ
  • Compresseur
  • LUFS Meter
  • Warmth
  • Limiter
  • Exports illimités en WAV, MP3, AIFF

Puis-je annuler mon achat ?

+

Comme les achats in-app sont gérés par Apple, vous pouvez demander un remboursement directement via App Store. Apple généralement accepte les demandes de remboursement dans les 14 jours suivant l'achat.

Pour plus d'informations, consultez App Store → Compte → Achats → Obtenir de l'aide.
